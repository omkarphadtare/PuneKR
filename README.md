# PuneKR
Android application as a Local Guide

# Overview
This application acts a Local Guide for "Pune" city.
It provides entire details regarding the city.

It offers services such as:

1)Accomodation

2)Transportation

3)Restaurants/Cafes

4)Tourist Places

5)Outskirts

6)Hospitals

7)Petrol Pumps

8)Police Stations

9)Medical Stores

10)Garages
